Installation

Console

Download the WiiMart WAD and the patched IOS WAD for your console type (Wii/vWii). We recommend YAWMME for installing WADs.

vWii is the Wii U's Wii Mode

If errors like -1022 or -1029 appear, uninstall the existing title first.

Dolphin

For those with Wii NAND backups

Install both the WiiMart WAD and the patched IOS WAD via Dolphin's Tools > Install WAD option.

For those without Wii NAND backups

Currently, getting the shop on a blank NAND has been unsuccessful.
